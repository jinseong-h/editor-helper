🚀 시작하기

이 앱은 영상 편집 작업을 체계적으로 관리하고, 수익과 시급을 자동 계산해주는 도구입니다.

1 프로필 → 고객 관리에서 작업할 유튜브 고객과 단가를 등록하세요.
2 작업 목록 → 새 작업 추가에서 편집 작업을 등록하세요.
3 스톱워치로 작업 시간을 기록하고, 완료 시 체크하면 시급이 자동 계산됩니다.

📋 작업 목록

  • 새 작업 추가 — 고객, 작업 종류(롱폼/숏폼/썸네일/기타), 완료 예상일, 영상 길이, 단가를 입력합니다. 단가는 고객 설정 기반으로 자동 계산됩니다.
  • 필터링 — 상단 필터바에서 미완료만 보기, 고객별, 종류별, 정렬 순서를 변경할 수 있습니다.
  • 스톱워치 — ▶ 버튼으로 작업 시간 측정을 시작하고, ⏸ 버튼으로 일시정지합니다. ↺ 버튼으로 초기화할 수 있습니다.
  • 시간 직접 수정 — 시간 표시(00:00:00)를 클릭하면 시간을 직접 입력할 수 있는 팝업이 나타납니다.
  • 영상 길이 입력 — 완성된 영상의 길이를 입력하면, 분당 단가 고객의 경우 단가가 자동 재계산됩니다.
  • 단가 수정 — ✏️ 아이콘을 눌러 수동 입력 모드로 전환하면 단가를 직접 수정할 수 있고, 🔒 아이콘을 누르면 자동 계산 모드로 돌아갑니다.
  • 작업 완료 — 하단의 "작업 완료" 체크박스를 누르면 완료 처리됩니다. 완료된 작업은 시급이 자동 계산됩니다.

📊 통계

  • 요약 — 완료된 작업 수, 총 수익, 총 작업 시간, 평균 시급을 한눈에 확인합니다.
  • 기간/고객 필터 — 월별 또는 전체 기간, 특정 고객을 선택하여 통계를 볼 수 있습니다.
  • 시급 차트 — 고객별 평균 시급을 바 차트로 비교합니다.
  • 작업시간 — 일별 작업 시간 추이를 막대 그래프로 확인합니다.
  • 주간 캘린더 — 몇 시부터 몇 시까지 어떤 작업을 했는지 주간 일정표 형태로 확인합니다.

📄 작업내역서

  • 고객별로 월간 작업내역서(인보이스)를 생성합니다.
  • 고객 선택 → 기간 선택(월별 또는 세부 기간 직접 선택) → 미리보기 확인 → PDF 다운로드
  • 입금 정보 — 은행명, 계좌번호, 예금주를 입력하면 작업내역서 하단에 포함됩니다. 정보를 저장해두고 불러올 수 있습니다.

👤 프로필 메뉴

  • 우측 상단 프로필 아이콘을 클릭하면 메뉴가 펼쳐집니다.
  • 고객 관리 — 작업할 유튜브 고객을 등록하고, 롱폼(건당/분당), 숏폼, 썸네일 단가를 설정합니다.
  • 설정 — PiP 타이머, 클라우드 동기화, 데이터 내보내기/불러오기를 관리합니다.
  • 로그아웃 — 현재 계정에서 로그아웃합니다.

💡 팁

  • 스톱워치는 여러 작업에서 동시에 실행할 수 있습니다.
  • 페이지를 닫거나 다른 기기를 사용해도 스톱워치는 멈추지 않고 계속 측정됩니다. (직접 ⏸ 버튼으로 멈춰야 합니다)
  • 자정을 넘기면 작업 시간은 자동으로 날짜별로 분할 기록됩니다.
  • 우측 하단 🌙/☀️ 버튼으로 다크모드를 전환할 수 있습니다.
  • 데이터를 정기적으로 내보내기하여 백업하는 것을 권장합니다.
